# PomodoroApp
An App which focuses on helping people to improve  focus and increase productivity
Developed the frontend as well as the backend of the app
● Pomodoro is a productivity technique in which a userworks
in cycles of 40+15 mins - 40 mins studies and 15 mins break
or 25+5mins. The app implements this approach by
allowing the user to set Pomodoro and get notified when he
has to take a break.
● The user can schedule his day and get notifications
accordingly, keeping track of how the other users are doing
in a leaderboard type format. Also shows various
motivational quotes - unlimited in number!
● TechStack-Android, Volley, Firebase
